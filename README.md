## 📦 Installation & Setup

To run the project locally:

```
git clone https://github.com/Techlyn-Solutions/Techlyn-Solutions-Website.git

cd Techlyn-Solutions-Website

npm install

npm run dev

Then open your browser and go to:

👉 http://localhost:3000
```
